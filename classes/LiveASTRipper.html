Attributes

steamroll [RW]

When true-valued, steamroll all ASTs created by the parser. Default is false. (Although referencing LiveASTRipper::Test will set it to true.)

Public Instance methods

parse (code)

Returns a line-to-AST hash where the AST corresponds to the method or block defined at the given line.

This method is the only requirement of a LiveAST parser plugin.
